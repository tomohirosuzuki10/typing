# typing — エンジニア向けタイピングアプリ

> エンジニア向けタイピングゲーム。プログラミング/コマンド用語で実用性も学べる。
---

## 1. プロジェクト概要

### 1-1. プロジェクト名
**typing
---

## 2. 機能要件

### Lv1（必須機能）
| # | 機能 | 説明 |
|---|---|---|
| 1 | ランダム問題表示 | 単語リストからランダムに1問選んで画面に表示 |
| 2 | 入力判定 | キーボード入力をリアルタイムで判定（1文字ずつ） |
| 3 | 正答数カウント | 正しく打ち切った問題数を表示 |
| 4 | タイマー | 残り時間カウントダウン（60秒など） |

### Lv2（拡張機能）
| # | 機能 | 説明 |
|---|---|---|
| 5 | スコア表示 | 正答数・問題数・正確率などの最終結果 |
| 6 | ミスタイプ数 | 間違ったキー入力の累計数 |
| 7 | WPM表示 | Words Per Minute（1分あたり打てた単語数） |

├─────────────────────────────────────────┤
│                                          │

### 3-2. 画面の状態（3パターン）

#### A. スタート画面
- タイトル
- 説明文
- 「スタート」ボタン
- ジャンル選択（任意）

#### B. プレイ中画面
- タイマー（カウントダウン）
- 正答数
- ミスタイプ数
- 現在の問題（大きく表示）
- 入力中の文字（色付きで進捗表示）

#### C. リザルト画面
- 最終スコア
- 正答数 / 総問題数
- ミスタイプ数
- WPM
- 正確率
- 「もう一度」ボタン

## 4. データ設計

### 4-1. 問題データ（words.js）

ジャンル別に配列で管理：

```javascript
const wordList = {
  command: [
    "git push", "npm install", "cd ..", "ls -la",
    "git commit", "docker run", "mkdir", "rm -rf",
    // ...30個くらい
  ],
  programming: [
    "function", "const", "return", "async", "await",
    "console", "import", "export", "class",
    // ...30個くらい
  ]
};
```

### 4-2. ゲーム状態（state）

```javascript
let state = {
  currentWord: "",      // 今表示してる問題
  typedText: "",        // 今まで打った文字
  correctCount: 0,      // 正答数
  missCount: 0,         // ミスタイプ数
  totalTyped: 0,        // 総入力文字数
  timeLeft: 60,         // 残り時間（秒）
  isPlaying: false,     // ゲーム中か
  startTime: null,      // 開始時刻
};
```

## 5. ロジック設計

### 5-1. ゲーム開始時の処理

```
1. state を初期化（スコアやミスを0に）
2. timeLeft を 60 にセット
3. isPlaying を true に
4. ランダムに1問選んで currentWord にセット
5. タイマー開始（1秒ごとに timeLeft を減らす）
6. キーボード入力を受け付け開始
7. 画面更新
```

### 5-2. キー入力時の処理

```
キーが押された
  ↓
押されたキー == 期待する次の文字？
  ↓                          ↓
 YES                         NO
  ↓                          ↓
typedText に追加              missCount++
totalTyped++                  画面に「ミス」表示
  ↓
typedText == currentWord ？
  ↓                  ↓
 YES                NO
  ↓                  ↓
正答数++           続きを入力
次の問題を出題     入力中の表示を更新
  ↓
画面更新
```

### 5-3. タイマー処理

```
setInterval で 1秒ごとに：
  timeLeft -= 1
  画面のタイマー表示を更新
  
  if timeLeft <= 0:
    ゲーム終了
    リザルト画面を表示
```

### 5-4. WPM計算

```
WPM = (正答数 / 経過時間（秒）) × 60

例：60秒で30問正解 → 30 / 60 × 60 = 30 WPM
```

正確率：
```
正確率 = (totalTyped - missCount) / totalTyped × 100
```
